توثيق واجهة برمجة تطبيقات المعارض

دليل شامل لاستخدام واجهة برمجة التطبيقات للمعارض

نظرة عامة

توفر واجهة برمجة التطبيقات هذه وظائف كاملة لإدارة المعارض، بما في ذلك إنشاء المعارض، الحصول على قائمة المعارض، تحديث تفاصيل المعرض، وحذف المعارض.

ملاحظة مهمة: تتطلب المسارات الإدارية صلاحيات مشرف (Admin) وتوكن JWT صالح.

عناوين URL الأساسية

المسارات العامة: https://visitsyria.fun/api/v1/new-events/exhibitions

المسارات الإدارية: https://visitsyria.fun/api/v1/admin/exhibitions

المسارات

GET /new-events/exhibitions

POST admin/exhibitions

إنشاء معرض

إنشاء معرض جديد مع عناوين باللغتين العربية والإنجليزية، الموقع، التاريخ، الوصف، وملفات الوسائط الاختيارية. يتطلب توثيق المشرف.

محتوى الطلب (multipart/form-data)

الحقل النوع مطلوب الوصف
placeAr نص نعم اسم المكان بالعربية
placeEn نص نعم اسم المكان بالإنجليزية
latitude عدد عشري نعم إحداثيات خط العرض لموقع المعرض
longitude عدد عشري نعم إحداثيات خط الطول لموقع المعرض
dateTime تاريخ ووقت نعم تاريخ ووقت المعرض
descriptionAr نص نعم وصف المعرض بالعربية
descriptionEn نص نعم وصف المعرض بالإنجليزية
officialSupporterAr نص لا اسم الداعم الرسمي بالعربية
officialSupporterEn نص لا اسم الداعم الرسمي بالإنجليزية
durationAr نص لا مدة المعرض بالعربية
durationEn نص لا مدة المعرض بالإنجليزية
cost عدد عشري لا تكلفة المعرض
targetAudienceAr نص لا الجمهور المستهدف بالعربية
targetAudienceEn نص لا الجمهور المستهدف بالإنجليزية
notesAr نص لا ملاحظات بالعربية
notesEn نص لا ملاحظات بالإنجليزية
classification نص لا تصنيف المعرض (art, history, science, culture, technology, literature, music, photography, crafts, food, fashion, nature, religion, sports, other)
classificationLabel كائن لا تسمية التصنيف باللغتين العربية والإنجليزية (يتم إنشاؤه تلقائيًا)
images ملف لا ملفات صور المعرض (JPEG, PNG, إلخ) - دعم لملفات متعددة

الاستجابة

تُرجع كائن المعرض الذي تم إنشاؤه.

Headers: Authorization: Bearer [admin-jwt-token]

Form Data:
