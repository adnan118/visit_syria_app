نظرة عامة

تتيح واجهة برمجة التطبيقات (API) الخاصة بالوسوم للمسؤولين إنشاء وتنظيم الوسوم التي يمكن استخدامها لتصنيف المنشورات والستوريز. تشمل الميزات:

  • إنشاء وتعديل الوسوم
  • عرض قائمة بالوسوم
  • الوصول إلى وسم محدد باستخدام المعرف أو الاسم المختصر
  • حذف الوسوم مع تنظيف المرجعيات تلقائياً

هيكل الوسم

كل وسم يحتوي على الحقول التالية:

الحقل النوع الوصف
id رقم المعرف الفريد للوسم
nameAr نص اسم الوسم بالعربية
nameEn نص اسم الوسم بالإنجليزية
slug نص الاسم المختصر للوسم (يُستخدم في الروابط)
createdAt تاريخ تاريخ الإنشاء
updatedAt تاريخ تاريخ آخر تحديث

إنشاء وسم

نقطة النهاية: POST /api/v1/tags

إنشاء وسم جديد (للمديرين فقط).

معلمات الطلب

الحقل النوع مطلوب الوصف
nameAr نص نعم اسم الوسم بالعربية
nameEn نص نعم اسم الوسم بالإنجليزية

مثال على الطلب

curl -X POST "https://visitsyria.fun/api/v1/tags" \
  -H "Authorization: Bearer <admin_access_token>" \
  -H "Content-Type: application/json" \
  -d '{
    "nameAr": "سياحة",
    "nameEn": "Tourism",
    "description": "المنشورات المتعلقة بالسياحة في سوريا"
  }'

مثال على الاستجابة

{
  "status": "success",
  "data": {
    "id": 123,
    "nameAr": "سياحة",
    "nameEn": "Tourism",
    "slug": "tourism",
    "createdAt": "2025-01-15T10:30:00.000Z",
    "updatedAt": "2025-01-15T10:30:00.000Z"
  }
}

عرض الوسوم

نقطة النهاية: GET /api/v1/tags

عرض قائمة بالوسوم.

مثال على الطلب

curl -X GET "https://visitsyria.fun/api/v1/tags" \
  -H "Authorization: Bearer <access_token>"

مثال على الاستجابة

{
  "status": "success",
  "data": [
    {
      "id": 123,
      "name": "سياحة",
      "slug": "tourism",
      "description": "المنشورات المتعلقة بالسياحة في سوريا",
      "color": "#3498db",
      "icon": "plane",
      "createdAt": "2025-01-15T10:30:00.000Z",
      "updatedAt": "2025-01-15T10:30:00.000Z"
    }
  ]
}

عرض وسم محدد

نقطة النهاية: GET /api/v1/tags/:id

عرض وسم محدد باستخدام المعرف أو الاسم المختصر.

مثال على الطلب

curl -X GET "https://visitsyria.fun/api/v1/tags/123" \
  -H "Authorization: Bearer <access_token>"

مثال على الاستجابة

{
  "status": "success",
  "data": {
    "id": 123,
    "name": "سياحة",
    "slug": "tourism",
    "description": "المنشورات المتعلقة بالسياحة في سوريا",
    "color": "#3498db",
    "icon": "plane",
    "createdAt": "2025-01-15T10:30:00.000Z",
    "updatedAt": "2025-01-15T10:30:00.000Z"
  }
}

تحديث وسم

نقطة النهاية: PUT /api/v1/tags/:id

تحديث وسم محدد (للمديرين فقط).

معلمات الطلب

الحقل النوع مطلوب الوصف
nameAr نص لا اسم الوسم بالعربية
nameEn نص لا اسم الوسم بالإنجليزية

مثال على الطلب

curl -X PUT "https://visitsyria.fun/api/v1/tags/123" \
  -H "Authorization: Bearer <admin_access_token>" \
  -H "Content-Type: application/json" \
  -d '{
    "nameAr": "سياحة سوريا",
    "nameEn": "Tourism Syria"
  }'

مثال على الاستجابة

{
  "status": "success",
  "data": {
    "id": 123,
    "nameAr": "سياحة سوريا",
    "nameEn": "Tourism Syria",
    "slug": "tourism-syria",
    "createdAt": "2025-01-15T10:30:00.000Z",
    "updatedAt": "2025-01-15T11:00:00.000Z"
  }
}

حذف وسم

نقطة النهاية: DELETE /api/v1/tags/:id

حذف وسم محدد (للمديرين فقط). يتم تنظيف المرجعيات تلقائياً من جميع المنشورات والستوريز.

مثال على الطلب

curl -X DELETE "https://visitsyria.fun/api/v1/tags/123" \
  -H "Authorization: Bearer <admin_access_token>"

مثال على الاستجابة

{
  "status": "success",
  "message": "Tag deleted successfully"
}
