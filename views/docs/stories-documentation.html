نظرة عامة

تتيح واجهة برمجة التطبيقات (API) الخاصة بالستوريز للمستخدمين إنشاء محتوى قصير-lived ومشاركته. تشمل الميزات:

  • إنشاء وتعديل الستوريز مع دعم الوسائط (صور ومقاطع فيديو)
  • عرض الستوريز مع تصفية حسب المستخدم
  • إدارة رؤية الستوريز (تنتهي تلقائياً بعد 24 ساعة)
  • الوصول إلى الستوريز الخاصة بي

هيكل الستوري

كل ستوري يحتوي على الحقول التالية:

الحقل النوع الوصف
author رقم معرف المؤلف
caption نص وصف الستوري
location نص الموقع الجغرافي
media مصفوفة الوسائط المرفقة (صور ومقاطع فيديو)
tags مصفوفة معرفات الوسوم المرتبطة بالستوري
expiresAt تاريخ تاريخ انتهاء الستوري (24 ساعة من الإنشاء)
views مصفوفة معرفات المستخدمين الذين شاهدوا الستوري
viewCount رقم عدد المشاهدات
createdAt تاريخ تاريخ الإنشاء
updatedAt تاريخ تاريخ آخر تحديث

إنشاء ستوري

نقطة النهاية: POST /api/v1/stories

إنشاء ستوري جديد مع دعم رفع الصور ومقاطع الفيديو.

معلمات الطلب

الحقل النوع مطلوب الوصف
caption نص لا وصف الستوري
location نص لا الموقع الجغرافي
tags مصفوفة أو نص لا معرفات الوسوم (مفصولة بفواصل أو مصفوفة)
images ملفات لا الصور المرفقة (يمكن رفع حتى 5 صور)
videos ملفات لا مقاطع الفيديو المرفقة (يمكن رفع حتى 5 مقاطع فيديو)

مثال على الطلب

curl -X POST "https://visitsyria.fun/api/v1/stories" \
  -H "Authorization: Bearer <access_token>" \
  -F "caption=وصف الستوري" \
  -F "location=دمشق" \
  -F "tags=1,2,3" \
  -F "images=@image1.jpg" \
  -F "videos=@video1.mp4"

مثال على الاستجابة

{
  "status": "success",
  "data": {
    "author": 456,
    "caption": "وصف الستوري",
    "location": "دمشق",
    "media": [
      {
        "type": "image",
        "file": "compressed-image1-1234567890.jpg"
      },
      {
        "type": "video",
        "file": "compressed-video1-1234567890.mp4"
      }
    ],
    "tags": [1, 2, 3],
    "expiresAt": "2025-01-16T10:30:00.000Z",
    "views": [],
    "viewCount": 0,
    "createdAt": "2025-01-15T10:30:00.000Z",
    "updatedAt": "2025-01-15T10:30:00.000Z"
  }
}

عرض الستوريز

نقطة النهاية: GET /api/v1/stories/feed

عرض الستوريز العامة من المستخدمين النشطين.

معلمات الاستعلام

الحقل النوع الافتراضي الوصف
page رقم 1 رقم الصفحة
limit رقم 20 عدد الستوريز في الصفحة

مثال على الطلب

curl -X GET "https://visitsyria.fun/api/v1/stories/feed?page=1&limit=10" \
  -H "Authorization: Bearer <access_token>"

مثال على الاستجابة

{
  "status": "success",
  "data": [
    // مصفوفة من الستوريز
  ],
  "pagination": {
    "currentPage": 1,
    "totalPages": 5,
    "totalStories": 45,
    "hasNext": true,
    "hasPrev": false,
    "pageSize": 10
  }
}

عرض ستوري محدد

نقطة النهاية: POST /api/v1/stories/:id/view

عرض ستوري محدد وزيادة عدد المشاهدات تلقائياً.

مثال على الطلب

curl -X POST "https://visitsyria.fun/api/v1/stories/123/view" \
  -H "Authorization: Bearer <access_token>"

مثال على الاستجابة

{
  "status": "success",
  "message": "Story viewed",
  "data": {
    "author": 456,
    "caption": "وصف الستوري",
    "location": "دمشق",
    "media": [
      {
        "type": "image",
        "file": "compressed-image1-1234567890.jpg"
      }
    ],
    "tags": [1, 2, 3],
    "expiresAt": "2025-01-16T10:30:00.000Z",
    "views": [],
    "viewCount": 25,
    "createdAt": "2025-01-15T10:30:00.000Z",
    "updatedAt": "2025-01-15T11:00:00.000Z"
  }
}

حذف ستوري

نقطة النهاية: DELETE /api/v1/stories/:id

حذف ستوري (للمؤلف فقط).

مثال على الطلب

curl -X DELETE "https://visitsyria.fun/api/v1/stories/123" \
  -H "Authorization: Bearer <access_token>"

مثال على الاستجابة

{
  "status": "success",
  "message": "Story deleted successfully"
}
