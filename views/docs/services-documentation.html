نظرة عامة

تتيح واجهة برمجة التطبيقات (API) الخاصة بالخدمات للمسؤولين إنشاء وتنظيم الخدمات التي يمكن استخدامها لتصنيف المنشورات. تشمل الميزات:

  • إنشاء وتعديل الخدمات
  • عرض قائمة بالخدمات
  • الوصول إلى خدمة محددة باستخدام المعرف
  • حذف الخدمات مع تنظيف المرجعيات تلقائياً

هيكل الخدمة

كل خدمة تحتوي على الحقول التالية:

الحقل النوع الوصف
id رقم المعرف الفريد للخدمة
nameAr نص اسم الخدمة بالعربية
nameEn نص اسم الخدمة بالإنجليزية
slug نص الاسم المختصر للخدمة (يُستخدم في الروابط)
createdAt تاريخ تاريخ الإنشاء
updatedAt تاريخ تاريخ آخر تحديث

إنشاء خدمة

نقطة النهاية: POST /api/v1/services

إنشاء خدمة جديدة (للمديرين فقط).

معلمات الطلب

الحقل النوع مطلوب الوصف
nameAr نص نعم اسم الخدمة بالعربية
nameEn نص نعم اسم الخدمة بالإنجليزية
slug نص لا الاسم المختصر (يُولد تلقائياً إذا لم يُقدم)

مثال على الطلب

curl -X POST "https://visitsyria.fun/api/v1/services" \
  -H "Authorization: Bearer <admin_access_token>" \
  -H "Content-Type: application/json" \
  -d '{
    "nameAr": "إرشاد سياحي",
    "nameEn": "Tourist Guidance"
  }'

مثال على الاستجابة

{
  "status": "success",
  "data": {
    "id": 123,
    "nameAr": "إرشاد سياحي",
    "nameEn": "Tourist Guidance",
    "slug": "tourist-guidance",
    "createdAt": "2025-01-15T10:30:00.000Z",
    "updatedAt": "2025-01-15T10:30:00.000Z"
  }
}

عرض الخدمات

نقطة النهاية: GET /api/v1/services

عرض قائمة بالخدمات.

مثال على الطلب

curl -X GET "https://visitsyria.fun/api/v1/services" \
  -H "Authorization: Bearer <access_token>"

مثال على الاستجابة

{
  "status": "success",
  "data": [
    {
      "id": 123,
      "name": "إرشاد سياحي",
      "slug": "tourist-guidance",
      "description": "خدمات الإرشاد السياحي في سوريا",
      "icon": "map",
      "createdAt": "2025-01-15T10:30:00.000Z",
      "updatedAt": "2025-01-15T10:30:00.000Z"
    }
  ]
}

عرض خدمة محددة

نقطة النهاية: GET /api/v1/services/:id

عرض خدمة محددة باستخدام المعرف.

مثال على الطلب

curl -X GET "https://visitsyria.fun/api/v1/services/123" \
  -H "Authorization: Bearer <access_token>"

مثال على الاستجابة

{
  "status": "success",
  "data": {
    "id": 123,
    "name": "إرشاد سياحي",
    "slug": "tourist-guidance",
    "description": "خدمات الإرشاد السياحي في سوريا",
    "icon": "map",
    "createdAt": "2025-01-15T10:30:00.000Z",
    "updatedAt": "2025-01-15T10:30:00.000Z"
  }
}

تحديث خدمة

نقطة النهاية: PUT /api/v1/services/:id

تحديث خدمة محددة (للمديرين فقط).

معلمات الطلب

الحقل النوع مطلوب الوصف
nameAr نص لا اسم الخدمة بالعربية
nameEn نص لا اسم الخدمة بالإنجليزية
slug نص لا الاسم المختصر

مثال على الطلب

curl -X PUT "https://visitsyria.fun/api/v1/services/123" \
  -H "Authorization: Bearer <admin_access_token>" \
  -H "Content-Type: application/json" \
  -d '{
    "nameAr": "إرشاد سياحي شامل",
    "nameEn": "Comprehensive Tourist Guidance"
  }'

مثال على الاستجابة

{
  "status": "success",
  "data": {
    "id": 123,
    "nameAr": "إرشاد سياحي شامل",
    "nameEn": "Comprehensive Tourist Guidance",
    "slug": "comprehensive-tourist-guidance",
    "icon": "map",
    "createdAt": "2025-01-15T10:30:00.000Z",
    "updatedAt": "2025-01-15T11:00:00.000Z"
  }
}

حذف خدمة

نقطة النهاية: DELETE /api/v1/services/:id

حذف خدمة محددة (للمديرين فقط). يتم تنظيف المرجعيات تلقائياً من جميع المنشورات.

مثال على الطلب

curl -X DELETE "https://visitsyria.fun/api/v1/services/123" \
  -H "Authorization: Bearer <admin_access_token>"

مثال على الاستجابة

{
  "status": "success",
  "message": "Service deleted successfully"
}
