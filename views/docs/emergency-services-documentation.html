Emergency Services API Documentation

This API provides endpoints for managing emergency services with support for multilingual titles (Arabic and English), phone numbers, and images.

Base URL

https://visitsyria.fun/api/v1/emergency-services

Endpoints

GET /emergency-services

Get All Emergency Services

Retrieve a list of all emergency services.

Response

Returns a JSON array of emergency service objects.

POST /emergency-services

Create Emergency Service

Create a new emergency service with Arabic and English titles, phone number, and optional image.

Request Body (multipart/form-data)

Field Type Required Description
titleAr string Yes Arabic title of the emergency service
titleEn string Yes English title of the emergency service
phoneNumber string Yes Phone number for the emergency service
images file No Image file for the emergency service (JPEG, PNG, etc.)

Response

Returns the created emergency service object.

Example Request

POST /emergency-services

Content-Type: multipart/form-data

Form Data:

  • titleAr: "خدمة الطوارئ"
  • titleEn: "Emergency Service"
  • phoneNumber: "+963123456789"
  • images: [image file]
GET /emergency-services/{id}

Get Emergency Service by ID

Retrieve a specific emergency service by its ID.

Parameters

Name Type Required Description
id integer Yes ID of the emergency service

Response

Returns the requested emergency service object.

PUT /emergency-services/{id}

Update Emergency Service

Update an existing emergency service.

Parameters

Name Type Required Description
id integer Yes ID of the emergency service to update

Request Body (multipart/form-data)

Field Type Required Description
titleAr string No Arabic title of the emergency service
titleEn string No English title of the emergency service
phoneNumber string No Phone number for the emergency service
images file No Image file for the emergency service (JPEG, PNG, etc.)

Response

Returns the updated emergency service object.

DELETE /emergency-services/{id}

Delete Emergency Service

Delete an emergency service by its ID.

Parameters

Name Type Required Description
id integer Yes ID of the emergency service to delete

Response

Returns a success message upon deletion.

Emergency Service Object

Example of an emergency service object returned by the API:

{
  "id": 1,
  "titleAr": "خدمة الطوارئ",
  "titleEn": "Emergency Service",
  "phoneNumber": "+963123456789",
  "image": "emergency-service-image.jpg",
  "createdAt": "2023-01-01T00:00:00.000Z",
  "updatedAt": "2023-01-01T00:00:00.000Z"
}

Error Responses

All error responses follow this format:

{
  "status": "error",
  "message": "Error description",
  "error": "Detailed error message"
}
